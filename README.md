# Social Media Analytics for Strategic Branding – Tesla 📱

## Project Overview
A data analytics project using Microsoft Excel and Google Sheets to analyze Tesla's social media engagement data, track performance KPIs, and deliver strategic branding recommendations based on data findings.

## Business Problem
Companies need to understand which content performs best on social media to maximize audience engagement and optimize their branding strategy. This project analyzes Tesla's social media data to identify top performing content types and posting patterns.

## Tools & Technologies
- **Microsoft Excel** — Pivot Tables, Advanced Charts, Data Analysis
- **Google Sheets** — Data Collection, Data Cleaning
- **Dataset** — Tesla social media engagement data across platforms

---

## Project Workflow

### Step 1: Data Collection & Cleaning (Google Sheets)
- Collected Tesla social media engagement data across platforms
- Removed duplicate records and standardized data formats
- Handled missing values and cleaned inconsistent entries
- Organized data by platform, content type, and date

### Step 2: Data Analysis (Excel)
- Built Pivot Tables to summarize engagement metrics by content type, platform, and time period
- Analyzed KPIs including:
  - Reach — number of unique users who saw the content
  - Impressions — total number of times content was displayed
  - Engagement Rate — interactions as a percentage of reach
  - Sentiment Trends — positive vs negative audience response over time

### Step 3: Visualization
- Created advanced Excel charts to visualize:
  - Engagement rate trends over time
  - Content type performance comparison
  - Platform-wise reach and impressions
  - Sentiment analysis over campaign periods

### Key Findings
- Identified content formats with the highest audience engagement
- Found optimal posting times and frequencies for maximum reach
- Identified sentiment patterns linked to specific campaigns or announcements

### Recommendations
- Focus content strategy on high-performing content formats identified in analysis
- Schedule posts during peak engagement windows
- Use sentiment trend data to guide messaging during product launches and announcements
